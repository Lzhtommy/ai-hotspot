<research_findings>
**Running the cron locally without invoking Trigger.dev Cloud**: The `runFetchSource` core function is deliberately runtime-independent (Plan 03 Task 1). The verification harness invokes it directly in a Node process — no Trigger.dev dev server required. This gives a fast deterministic feedback loop without having to manually click "run" in the Trigger.dev dashboard.

**Why not invoke `fetchSource.run()` directly as a Trigger.dev task?** Task handles exposed by `@trigger.dev/sdk` only expose a `.trigger()` dispatch method to the Trigger.dev Cloud API — not a local `.run()` callable. Running the task body requires the Trigger.dev dev worker process. For verification we want a pure local run; calling `runFetchSource` accomplishes that. The UAT document covers the Trigger.dev dashboard path separately.

**Simulating "broken source"** (SC #2): insert a temporary source with `rssUrl = '/__nonexistent_route_for_phase2_verification__'`. `fetchRSSHub` will get a 404 from the RSSHub upstream, which maps to `RSSHubError` in `src/lib/rsshub.ts`. `runFetchSource` catches the error and increments `consecutive_error_count`. The harness asserts: (a) that source's error_count increased, (b) the OTHER seeded sources have ok status. The broken source is removed after verification.

**Timestamp UTC check (SC #4)**: query one item where `published_at_source_tz IS NOT NULL`; assert `published_at` is a TIMESTAMPTZ whose instant matches `published_at_source_tz` interpreted as an absolute time. Because Postgres TIMESTAMPTZ is internally UTC, any successful insert automatically satisfies this — but we confirm programmatically for documentation value.
</research_findings>

    Flow:
    1. **Pre-flight**: Assert DATABASE_URL is set; assert ≥3 active sources exist (seeded by Plan 04); assert `items.published_at_source_tz` column exists.
    2. **Insert broken source**: INSERT a sentinel source `{name: 'BROKEN (verify-ingest sentinel)', rss_url: '/__verify_ingest_broken_sentinel__', is_active: true, language: 'en', weight: '0.1'}` with ON CONFLICT DO NOTHING. Record its id.
    3. **Run 1**: Enumerate active sources, call `runFetchSource` for each, collect `FetchSourceResult[]`. Record timestamp T1.
    4. **Snapshot 1**: Query `SELECT COUNT(*) FROM items`. Call this N1.
    5. **Run 2** (immediately): Same enumeration + `runFetchSource` for each. Record timestamp T2.
    6. **Snapshot 2**: `SELECT COUNT(*) FROM items`. Call this N2.
    7. **Criterion #1 (idempotency)**: Assert `N2 == N1` (exactly zero new items on the second run). Also assert `SELECT COUNT(*) FROM (SELECT url_fingerprint, COUNT(*) c FROM items GROUP BY url_fingerprint HAVING COUNT(*)>1)` is 0 (no dup fingerprints).
    8. **Criterion #2 (source isolation)**: Assert the broken sentinel source has `consecutive_error_count >= 1` AND at least one non-broken source has `status='ok'` observable via a successful run 1 result AND `items` has rows with `source_id` matching the non-broken source(s).
    9. **Criterion #3 (counter accuracy per D-08)**:
       - For the broken sentinel: `consecutive_error_count >= 1`, `last_fetched_at IS NULL` (never updated on error), `consecutive_empty_count == 0` (untouched).
       - For each non-broken source whose run 1 returned `newCount>=1` (ok with new items): `consecutive_empty_count == 0`, `consecutive_error_count == 0`, `last_fetched_at IS NOT NULL`.
       - For each non-broken source whose run 2 returned `newCount==0` after run 1 inserted items: `consecutive_empty_count == 1` (run 2 incremented from 0), `consecutive_error_count == 0`, `last_fetched_at` updated to T2.
    10. **Criterion #4 (UTC storage)**: `SELECT published_at, published_at_source_tz FROM items WHERE published_at_source_tz IS NOT NULL LIMIT 5`. For each row, parse `published_at_source_tz` as a Date; assert it equals `published_at` (±1 second tolerance). If zero rows with source_tz (possible if all seeds produced Z-offset), log a warning and skip this row-level check (the schema column existence is the primary criterion).
    11. **Cleanup**: DELETE the broken sentinel source and any items inserted against it (by source_id).
    12. **Summary**: Print PASS/FAIL per criterion.

    Exit code: 0 if all criteria PASS; 1 if any FAIL. Any thrown exception during the harness itself also exits 1 with stack trace.

  <action>
    Create `scripts/verify-ingest.ts`:

    ```typescript
    /**
     * Phase 2 verification harness — exercises the ingestion pipeline end-to-end
     * against a live Neon dev branch + live RSSHub and asserts all 4 Phase 2
     * success criteria from ROADMAP.md.
     *
     * Run: `pnpm verify:ingest`
     *
     * Exits 0 on PASS, 1 on any failure. Cleans up its own sentinel source on exit.
     *
     * NOT run by CI — requires live RSSHub reachability + live Neon dev branch.
     */
    import 'dotenv/config';
    import { config } from 'dotenv';
    config({ path: '.env.local' });

    import { and, eq, inArray, sql } from 'drizzle-orm';
    import { db } from '../src/lib/db/client';
    import { items, sources } from '../src/lib/db/schema';
    import { runFetchSource } from '../src/lib/ingest/fetch-source-core';

    const BROKEN_ROUTE = '/__verify_ingest_broken_sentinel__';
    const BROKEN_NAME = 'BROKEN (verify-ingest sentinel)';

    type CriterionResult = { name: string; pass: boolean; detail: string };

    const results: CriterionResult[] = [];

    function record(name: string, pass: boolean, detail: string) {
      results.push({ name, pass, detail });
      console.log(`[${pass ? 'PASS' : 'FAIL'}] ${name}: ${detail}`);
    }

    async function insertBrokenSentinel(): Promise<number> {
      await db
        .insert(sources)
        .values({ name: BROKEN_NAME, rssUrl: BROKEN_ROUTE, language: 'en', weight: '0.1' })
        .onConflictDoNothing({ target: sources.rssUrl });
      const found = await db.select({ id: sources.id }).from(sources).where(eq(sources.rssUrl, BROKEN_ROUTE));
      if (!found[0]) throw new Error('failed to insert broken sentinel');
      return found[0].id;
    }

    async function cleanup(brokenId: number) {
      await db.delete(items).where(eq(items.sourceId, brokenId));
      await db.delete(sources).where(eq(sources.id, brokenId));
    }

    async function enumerateActive() {
      return db
        .select({ id: sources.id, rssUrl: sources.rssUrl, name: sources.name })
        .from(sources)
        .where(eq(sources.isActive, true));
    }

    async function main() {
      if (!process.env.DATABASE_URL) throw new Error('DATABASE_URL not set');

      console.log('== Pre-flight ==');
      const preActive = await enumerateActive();
      if (preActive.length < 3) {
        throw new Error(`Expected >=3 active sources (run pnpm db:seed first). Found ${preActive.length}.`);
      }
      console.log(`Pre-flight OK: ${preActive.length} active sources.`);

      // Insert broken sentinel
      const brokenId = await insertBrokenSentinel();
      console.log(`Broken sentinel source id=${brokenId}`);

      try {
        const active = await enumerateActive();

        console.log('\n== Run 1 ==');
        const run1Start = Date.now();
        const run1Results = [];
        for (const s of active) {
          const r = await runFetchSource({ sourceId: s.id, rssUrl: s.rssUrl });
          console.log(`  [${r.status}] source=${s.name} new=${r.newCount} seen=${r.seenCount}${r.errorKind ? ' err=' + r.errorKind : ''}`);
          run1Results.push({ ...r, name: s.name });
        }
        const n1Row = await db.execute(sql`SELECT COUNT(*)::int AS n FROM items`);
        const N1 = (n1Row as unknown as Array<{ n: number }>)[0].n;
        console.log(`Items after Run 1: ${N1}`);

        console.log('\n== Run 2 (idempotency check) ==');
        const run2Results = [];
        for (const s of active) {
          const r = await runFetchSource({ sourceId: s.id, rssUrl: s.rssUrl });
          console.log(`  [${r.status}] source=${s.name} new=${r.newCount} seen=${r.seenCount}${r.errorKind ? ' err=' + r.errorKind : ''}`);
          run2Results.push({ ...r, name: s.name });
        }
        const n2Row = await db.execute(sql`SELECT COUNT(*)::int AS n FROM items`);
        const N2 = (n2Row as unknown as Array<{ n: number }>)[0].n;
        console.log(`Items after Run 2: ${N2}`);

        console.log('\n== Assertions ==');

        // Criterion #1: idempotency — N2 must equal N1; zero duplicate fingerprints.
        const dupRow = await db.execute(sql`
          SELECT COUNT(*)::int AS n FROM (
            SELECT url_fingerprint FROM items GROUP BY url_fingerprint HAVING COUNT(*) > 1
          ) q
        `);
        const dupCount = (dupRow as unknown as Array<{ n: number }>)[0].n;
        record(
          'SC#1 idempotency',
          N2 === N1 && dupCount === 0,
          `N1=${N1}, N2=${N2}, dup-fingerprint-groups=${dupCount}`,
        );

        // Criterion #2: source isolation — broken sentinel has an error; at least one other succeeded.
        const brokenRow = await db.select().from(sources).where(eq(sources.id, brokenId));
        const okCount = run1Results.filter((r) => r.status === 'ok' && r.sourceId !== brokenId).length;
        record(
          'SC#2 source isolation',
          (brokenRow[0]?.consecutiveErrorCount ?? 0) >= 1 && okCount >= 1,
          `broken.err_count=${brokenRow[0]?.consecutiveErrorCount}, ok_non_broken=${okCount}/${active.length - 1}`,
        );

        // Criterion #3: counter semantics per D-08.
        const allSources = await db.select().from(sources).where(inArray(sources.id, active.map((s) => s.id)));
        const brokenSrc = allSources.find((s) => s.id === brokenId);
        let sc3Ok = true;
        const sc3Detail: string[] = [];
        if (!brokenSrc) {
          sc3Ok = false;
          sc3Detail.push('broken source row missing');
        } else {
          if (brokenSrc.lastFetchedAt !== null) {
            sc3Ok = false;
            sc3Detail.push(`broken.last_fetched_at must be null on error, got ${brokenSrc.lastFetchedAt}`);
          }
          if (brokenSrc.consecutiveEmptyCount !== 0) {
            sc3Ok = false;
            sc3Detail.push(`broken.empty_count must be 0 (untouched on error), got ${brokenSrc.consecutiveEmptyCount}`);
          }
          if (brokenSrc.consecutiveErrorCount < 1) {
            sc3Ok = false;
            sc3Detail.push(`broken.error_count must be >=1, got ${brokenSrc.consecutiveErrorCount}`);
          }
        }
        // Non-broken sources that succeeded in run1 with new items: counters both 0, last_fetched_at set.
        for (const r of run1Results) {
          if (r.sourceId === brokenId) continue;
          if (r.status !== 'ok') continue;
          const src = allSources.find((s) => s.id === r.sourceId);
          if (!src) continue;
          if (src.lastFetchedAt === null) {
            sc3Ok = false;
            sc3Detail.push(`${src.name}.last_fetched_at should be set after successful run`);
          }
          if (src.consecutiveErrorCount !== 0) {
            sc3Ok = false;
            sc3Detail.push(`${src.name}.error_count should be 0 after success, got ${src.consecutiveErrorCount}`);
          }
        }
        record('SC#3 counter accuracy', sc3Ok, sc3Detail.join('; ') || 'all counters match D-08');

        // Criterion #4: UTC storage + source_tz preservation.
        const tzRows = (await db.execute(sql`
          SELECT published_at, published_at_source_tz
          FROM items
          WHERE published_at_source_tz IS NOT NULL
          LIMIT 5
        `)) as unknown as Array<{ published_at: string; published_at_source_tz: string }>;

        let sc4Ok = true;
        const sc4Detail: string[] = [];
        if (tzRows.length === 0) {
          sc4Detail.push('no items with non-null published_at_source_tz yet — schema check only');
        } else {
          for (const row of tzRows) {
            const utcMs = new Date(row.published_at).getTime();
            const srcTzMs = new Date(row.published_at_source_tz).getTime();
            const deltaMs = Math.abs(utcMs - srcTzMs);
            if (deltaMs > 1000) {
              sc4Ok = false;
              sc4Detail.push(`utc=${row.published_at} vs source_tz=${row.published_at_source_tz} delta=${deltaMs}ms`);
            }
          }
          if (sc4Ok) sc4Detail.push(`${tzRows.length} rows checked, all match within 1s`);
        }
        // Column existence sanity — always checked.
        const colRow = (await db.execute(sql`
          SELECT data_type, is_nullable FROM information_schema.columns
          WHERE table_name = 'items' AND column_name = 'published_at_source_tz'
        `)) as unknown as Array<{ data_type: string; is_nullable: string }>;
        if (colRow.length !== 1 || colRow[0].data_type !== 'text' || colRow[0].is_nullable !== 'YES') {
          sc4Ok = false;
          sc4Detail.push(`column shape incorrect: ${JSON.stringify(colRow)}`);
        }
        record('SC#4 utc storage + source_tz', sc4Ok, sc4Detail.join('; '));

        // Final summary
        console.log('\n== Summary ==');
        const failed = results.filter((r) => !r.pass);
        for (const r of results) console.log(`  [${r.pass ? 'PASS' : 'FAIL'}] ${r.name}`);
        if (failed.length > 0) {
          console.error(`\n${failed.length}/${results.length} criteria FAILED.`);
          process.exit(1);
        }
        console.log(`\nAll ${results.length} criteria PASSED.`);
      } finally {
        console.log('\n== Cleanup ==');
        await cleanup(brokenId);
        console.log(`Removed broken sentinel source id=${brokenId} and its items.`);
      }
    }

    main().catch((e) => {
      console.error('VERIFY FAILED:', e);
      process.exit(1);
    });
    ```

    Notes on the `db.execute(sql\`...\`)` pattern: Drizzle's HTTP driver returns raw rows as `unknown`; cast per-row for the column subsets we need. Do NOT reach for a second driver — reuse the existing `db` singleton.
